# Roadmap

The active migration plan lives in the repository root at `MIGRATION_PLAN.md`.
Parity tracking lives in `PARITY_MATRIX.md`.

Post-1.0 priorities:

1. Continue algorithmic optimization for the mesh, tetrahedron, and cylinder-segment families.
2. Expand prepared-state and outer-loop JIT ergonomics for optimization workloads.
3. Keep parity and profiling dashboards tight as upstream Magpylib evolves.
4. Add more end-to-end examples, benchmark case studies, and release notes as the project grows.
